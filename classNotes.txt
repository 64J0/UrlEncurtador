### Pacotes npm utilizados:

* express
* mongoose
* ejs
* nodemon
* shortid

### Algumas especificações:

O pacote ejs será usado para renderizar as páginas HTML do projeto.
Rota '/' -> Renderiza a tela com um form contendo um h1, um label, um input e um button. E embaixo será apresentada uma table com 3 colunas (Full Url, Short Url, e Clicks).
Para estilizar o site será usado o Bootstrap.

Rota '/shortUrls' -> Conecta com o banco de dados e salva uma nova Url.

Rota '/:shortUrl' -> Recupera um valor do banco de dados, correspondente ao valor enviado nos parâmetros da requisição.

Modelo do banco de dados: urlShortener { full, short, clicks }.

### Comando para iniciar o servidor em modo de desenvolvimento (nodeon):

npm run devStart
